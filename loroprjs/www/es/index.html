Proyectos Loro de demostración
Para instalar cualquiera de estos proyectos en tu sistema LoroEDI:
1) Utiliza el comando "Instalar..." (Ctrl-I);
2) Escribe la dirección de esta página en la sección de proyecto "externo";
3) Utiliza el comando "Navegar" (Alt-N);
4) Haz un clic en el enlace del proyecto a instalar;
5) Acepta el cuadro de diálogo para completar la instalación.
Así de fácil!
