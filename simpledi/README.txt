2003-05-24 (0.7.5)

	- Creado este subproyecto de lo que era el anterior entorno de desarrollo
	  de Loro, versión 0.7.x. Para esto se trajeron los elementos no utilizados
	  de loroedi 0.8pre12.
	  
	  Se crea este subproyecto principamente para fines de backup.
	  
	  N O T A S
	  	- Al parecer, el viejo entorno sigue funcionando bien PERO NO se hace
		  ahora ningún esfuerzo por hacerle mantenimiento!
		- Se requiere el nuevo loroedi.jar ;-)
		- La ayuda general (F1) y los mensajes sobre versión aluden al nuevo 
		  entorno loroedi.
	  
	- Para compilar:
	  		ant
	  Deja ejecutable simpledi.jar en ${build}.  
	  Ver build.properties. 
	  
	- Para ejecutar:
		Poner simpledi.jar en el mismo directorio en donde se encuentre en donde
		esté loroedi.jar bajo la instalación del sistema. 
		y ejecutar:
			java -jar simpledi.jar
