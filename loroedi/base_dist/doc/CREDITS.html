Componentes de software incluidos:
  • BeanShell. Para implementación de algoritmos en Java.
  • JavaCC. Clases para el análisis léxico/sintáctico del derivador.
  • JEdit. Componente de edición con resaltado de sintaxis.
  • JGraph. Componente para diagramación.
  • IzPack. Módulo instalador del sistema.
  • Iconos. Algunos íconos genéricos de Sun Microsystems.
  • Kunststoff L&F, SkinLF. Manejo de LookAndFeel.


Herramientas de desarrollo utilizadas:

  • Java. Compilación de los fuentes en Java.
  • JEdit. Editor para desarrollo.
  • Apache Ant. Contrucción del sistema.
