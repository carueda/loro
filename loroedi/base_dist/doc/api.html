Documentación para desarrollo (API*)
*API, Application Programming Interface, es el término estándar.

  • ¿Qué se tiene?
  • ¿para qué sirve?
  • ¿cómo se usa?
  • ¿bajo qué condiciones?
son algunas de las consideraciones claves que se presentan permanentemente en la práctica de la programación.

Hoy en día, "reinventar la rueda" en asuntos de programación es una tarea que difícilmente emprenda un programador (salvo en casos muy particulares). De hecho, el software se contruye por lo general sobre software ya existente, así que se vuelve imprescindible contar con toda la información posible que resuelva las preguntas de arriba.

No sólo el lenguaje Loro como tal contempla este importante aspecto incluyendo los elementos sintácticos para incluir la documentación a nivel de fuentes, sino que, a nivel del entorno integrado, también genera automáticamente una versión HTML de dicha documentación, lo que trae consigo las facilidades de "navegación" que ayudan en el trabajo. Desde luego, el "contenido" de dicha documentación sigue siendo una responsabilidad del programador.
