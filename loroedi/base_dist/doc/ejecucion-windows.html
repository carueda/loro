Ejecución en Windows

Utilizando el explorador de archivos de Windows, apunta con el ratón y hazle "doble-clic" a:
${DIR}\bin\loroedi.bat

lo cual lanzará el Entorno de Desarrollo Integrado de Loro (LoroEDI).
 

Nota: Es posible que el explorador de archivos esté ocultando la extensión de los archivos, así que se verá sólamente loroedi.
Sugerencia: Crea un "acceso directo" (shortcut) para facilitar el uso rutinario posterior de LoroEDI.  Una manera fácil de lograrlo es "arrastrar" con el ratón el archivo indicado arriba hasta el lugar de tu preferencia, por ejemplo al escritorio, o a la barra de tareas contigua al menú de Inicio. Puedes encontrar más detalles en la documentación de Windows y/o utilizando la opción de ayuda en línea del sistema operativo.

Otras formas alternativas de lanzar el LoroEDI

2a. forma alternativa
* Abre una ventana de comandos (DOS).
* Teclea las siguientes órdenes (cada una terminada con la tecla Intro o Enter):
    cd ${DIR}\bin
    loroedi.bat

3a. forma alternativa
En esta instalación también se ha transferido:
${DIR}\bin\loroedi.jar

que puede ser ejecutado directamente en muchos sistemas operativos, incluyendo Windows (recuerda que es posible no ver la extensión). Así que puedes intentar hacerle "doble-clic". Si efectivamente esto funciona en tu sistema, quizá quieras seguir la sugerencia comentada arriba sobre "acceso directo" pero esta vez aplicada sobre loroedi.jar.
