Intérprete Interactivo

A través del intérprete interactivo se pueden ejecutar fragmentos de código de manera inmediata. Esto incluye el lanzamiento de programas completos (haciendo invocación de algoritmos), así como hacer declaraciones de variables y ejecución de instrucciones.

Esta característica se constituye en un excelente medio para conocer el lenguaje.

En general, todo lo que se escriba al frente del indicador $ se toma como código Loro. La excepción a esta regla es cuando el primer caracter es un punto ('.'), en cuyo caso se interpreta como comando dirigido al propio intérprete. Estos comandos son:

      .?          - Muestra esta ayuda
      .limpiar    - Limpia la ventana
      .vars       - Muestra las variables declaradas hasta el momento
      .borrar ID  - Borra la declaración de la variable indicada
      .borrarvars - Borra todas las variables declaradas
      .??         - Muestra otros comandos avanzados

      .modo         - Muestra el modo de interpretación actual.
                      Hay dos modos de operación:
                        - ejecución completa (por defecto)
                        - sólo compilación
      .cambiarmodo  - Intercambia el modo de interpretación
      .verobj nivel - Pone maximo nivel para visualizar objetos
      .verarr long  - Pone maxima longitud para visualizar arreglos
