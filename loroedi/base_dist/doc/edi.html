El entorno Loro

El entorno de trabajo está basado en proyectos. Para cada proyecto en el que estés trabajando vas a tener una ventana de interacción del siguiente estilo:

El elemento central es una diagrama que te muestra los componentes actuales de tu proyecto. En el ejemplo aparecen: una especificación y tres algoritmos, dos de los cuales resultan ser implementaciones para solucionar la especificación. Desde esta ventana llevas a cabo las operaciones posibles sobre tu proyecto, como la creación y modificación de los componentes, la compilación, la visualización de la documentación y la ejecución de algoritmos, entre otros.

Cuando abres para edición uno de los componentes de tu proyecto, se abre una nueva ventana del siguiente estilo:

Desde aquí puedes llevar a cabo acciones que afectan particularmente al componente.

Mecanismos de operación
Para interactuar con el sistema, pueden utilizarse las opciones incluidas en la barra de menúes, la barra de herramientas, y el teclado. Los elementos del diagrama también tienen asociados menúes "popup" con opciones específicas. Un menú popup aparece haciéndo clic con el botón derecho del ratón sobre el elemento correspondiente.

Acciones generales

  • Abrir la ventana de documentación.
  • Abrir el Intérprete Interactivo.
  • Abrir la ventana de información general.
Acciones para acceder a proyectos
  • Abrir un proyecto ya existente en espacio de trabajo.
  • Instalar un proyecto para ponerlo en el espacio de trabajo.
Acciones sobre el proyecto de trabajo
  • Compilar el proyecto completo.
  • Importar programas del exterior al proyecto.
  • Exportar el proyecto hacia el exterior.
  • Agregar/Eliminar un componente al proyecto.
Acciones sobre una unidad
Una unidad es una especificación, un algoritmo o una clase.
  • Editar el código fuente de la unidad.
  • Compilar la unidad .
  • Visualizar la documentación de la unidad.
  • Ejecutar la unidad, cuando ésta es un algoritmo.
