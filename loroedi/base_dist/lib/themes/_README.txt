En este directorio se buscan themes para Look&Feel para 
usarse con el paquete skinlf.jar incluido con la distribución
de Loro. Por ejemplo, si aquí se encuentra el theme
aquathemepack.zip, entonces puede definirse la preferencia 
correspondiente así:

loroedi.pref.laf=aquathemepack.zip

Esta línea debe aparecer en el archivo loroedi.pref que se
encuentra bajo el directorio Loro del usuario.
